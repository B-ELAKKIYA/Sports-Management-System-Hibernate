The Community Sports Ground Management System allows administrators to:
Register ground bookings
Confirm or cancel bookings
Calculate booking amounts
Track actual slot usage
Record overtime and ground condition feedback
View booking and usage records

This project uses:
Java
Hibernate ORM 6
Oracle Database
HikariCP Connection Pool

Technologies Used
Java 17+
Hibernate ORM 6.4.4
Oracle 11g/19c
Maven (if applicable)
JDBC
HikariCP
